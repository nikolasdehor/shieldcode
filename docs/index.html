ShieldCode is a Claude Code skill that enforces OWASP Top 10 prevention and production-grade error handling in every line Claude generates.

./install.sh

What it prevents

SQL Injection

Enforces parameterized queries for every database call. No string concatenation with user input, ever.

XSS Attacks

Enforces safe DOM APIs, textContent over innerHTML, CSP headers, and Jinja2 autoescape.

Weak Authentication

Requires bcrypt/argon2 for passwords. Full JWT verification with signature, expiration, and issuer. Rate-limited endpoints.

Broken Authorization

Server-side role checks from verified JWT. Resource ownership validated before every modification.

Exposed Secrets

No hardcoded API keys or passwords. Environment variable validation at startup. Secrets never logged or returned in responses.

Silent Errors

No empty catch blocks. Correct HTTP status codes. Stack traces never sent to clients. Structured logging with correlation IDs.

Path Traversal

File paths resolved and validated against the upload directory. Upload types and sizes restricted.

Slopsquatting

Only real, verified packages suggested. Includes a trusted package reference list for both Node.js and Python.

How it works

1

Install the skill

Run ./install.sh to copy SKILL.md to ~/.claude/skills/shieldcode/.

2

Claude loads the skill automatically

Claude Code reads all skills from ~/.claude/skills/ and applies them to every session.

3

Rules activate on relevant code

When writing code that handles user input, authentication, database queries, APIs, or error handling, Claude applies ShieldCode's rules automatically.

4

Secure code, every time

No prompting required. No configuration. Claude generates parameterized queries, bcrypt hashing, ownership checks, and structured error handling by default.

Example

auth.ts 
// Without ShieldCode - Claude might generate this:
const hash = crypto.createHash('sha256').update(password).digest('hex');
const user = await db.query(`SELECT * FROM users WHERE email = '${email}'`);

// With ShieldCode - Claude generates this:
const hash = await bcrypt.hash(password, 12);
const user = await db.query('SELECT * FROM users WHERE email = $1', [email]);
